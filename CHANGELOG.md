# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-01-01

### Added
- Interactive CLI built with `questionary` and `rich`
- Jinja2 template engine for README generation
- Four project templates: Python, Machine Learning, Web, Java
- Input validation for all user-facing fields
- Overwrite protection with user confirmation
- In-terminal Markdown preview before saving
- Custom output path support
- Unit tests for validator, template engine, and file manager
- GitHub Actions CI/CD pipeline
- Support for Python 3.10, 3.11, and 3.12
